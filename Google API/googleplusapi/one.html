follow button

extra attributes to follow button
  • class (g-follow) Specify g-follow to render the follow button in HTML5-valid syntax. Otherwise, use the <g:follow/> markup.
  • data-href (URL)
    link to the Google+ page or user profile
    ex->https://plus.google.com/110967630299632321627
    ->https://plus.google.com/+LarryPage
  • data-annotation (none bubble vertical-bubble)
  • data-height (15 20 24)
    pixel height of button
